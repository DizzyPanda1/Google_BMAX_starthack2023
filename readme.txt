To start the programme run in the terminal "streamlit run main.py" that will create a local server on port 8051 that will allow to access the demo
Streamlit library might not be installed already, in that case  "pip install streamlit" or "conda install streamlit" should solve it.

Google cloud Api limits the api accesses of apps that are not verified, therefore the only email that have access to it are those of the members of the team.
